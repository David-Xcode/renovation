抽油烟机RANGE HOOD · 中式爆炒第一件

市场是方太系(含华帝)、老板、美的系三足鼎立，TOP3 合计约占近七成，5000 元以上高端段被方太、老板把控。 现行国标 GB/T 17713-2022(2024-01-01 实施)把油脂分离度门槛提到 ≥85%、引入更贴近实际工况的"工作风量"。 核心矛盾：风量、风压/静压、噪音三者互相牵制——单看风量极易被参数误导，高层 / 共用烟道尤其要看静压。高置信

⌖ 怎么花钱最值

先按人口定容量，再谈品牌：20–25L 够 1–3 人、25–30L 适合 3–5 人（最常见）、50–60L 一体机留给大家庭或要烤大件——买大占柜、买小塞不下烤盘。性价比首选国产：美的、格兰仕覆盖大多数真实需求，¥3000–6000 能买到 34L 级带速蒸+自净的成熟一体机，是花得最值的甜点。追求加热均匀上松下变频；要德系整体厨房美观选西门子 / 博世；只有对做工耐用有执念且预算充足才上 Miele。最大误区是为"一机替代三机"多花钱——重度烘焙 / 蒸制者，一体机纯蒸 / 纯烤常不如同价位专业单品，拆开买"微波+独立蒸烤"更划算。智能 / AI 款溢价约 30–50%，按是否真用决定。

安装安全规范（综述）INSTALL SAFETY · 把好电气/燃气/散热/固定四道关

嵌入式厨电的安装安全，本质是把好电气、燃气、散热、固定四道关。多数具体数值为厂商说明书或安装经验值， 施工必须以购买型号的说明书为准，装修前定型号、预留开孔与电源——后期几乎无法补救。

⚡
电气：独立专用回路大功率嵌入厨电（蒸烤、微蒸烤整机约 2000–3000W）用独立带地专用回路、16A、线径 ≥2.5mm²、220V~50Hz，不与其他大功率共用，建议配漏电保护(RCD/RCBO)。
高置信
🔌
插座：侧方、可检修、避散热区插座预留在机身侧方相邻柜内、便于检修处，避开正后方散热 / 走线空间与高温，数量留余量。
高置信
📐
开孔：先定型号、按手册 niche 尺寸开孔因型号而异（蒸烤机身常约 595×456×542mm、微蒸烤约 595×455×545mm），柜体开孔须比机身每侧多留约 2–5cm，以手册壁龛(niche)尺寸为准，严禁套用通用值。
高置信
🔥
燃气：符合 CJJ 12 净距，持证施工涉燃气须符合 CJJ 12-2013 及 GB 16410 / GB 17905：灶墙净距 ≥10cm、距木质家具 ≥20cm、橡胶软管 ≤2m 无接头不穿墙；通常须由持证燃气公司施工，业主不可自行改管，装后验燃气接口无泄漏、排烟通畅、不正对燃气热水器排烟口。
中置信
🛡️
油烟机：止逆阀双重 + 管径匹配烟管接公共烟道处机身自带+烟道口独立蝶阀双重止逆，接口密封圈 / 防火泥封堵；管径与接口（150 / 160 / 180mm）匹配，严禁缩径，减少 90° 弯头。
高置信
🔩
固定：承重柜体、防跌落 + 认 3C嵌入电器固定在能承重(经验值 ≥60kg)的柜体 / 支架并保持水平，膨胀螺栓 / 层板到位防振动松动；微波炉 / 电烤箱属 CCC(3C) 强制认证，认准 3C 标识。
高置信

隔热与散热规范（综述）HEAT INSULATION · 前置散热是嵌入刚需

散热做不好，柜体受热变形开裂、电器积热降寿甚至起火。原则是 前置散热 + 留足间隙 + 橱柜通风 + 耐热柜体 + 远离热源。下列间隙多为厂商 / 经验值，以型号说明书为准。

➡️
前置散热是嵌入刚需嵌入式专用机为前置散热（散热口在正面下方），热风经专用风道与前板散热孔排出；普通台式机多上 / 后散热，不可直接塞进密闭定制柜，须用匹配 trim kit 或选嵌入专用机。
高置信
🌡️
散热间隙经验值微波 / 微蒸烤：顶部约 5–10cm、两侧各 2–3cm、背部 3–5cm；蒸烤箱后部约 ≥35–45mm（方太口径）。散热口严禁封死。
经验值
🪟
柜体通风口 ≥250cm²电器柜开足够通风口（经验值 ≥250cm²）且不被封板 / 台面遮盖，保证进排风对流；可在背板 / 底板开散热孔或用透气材料，关机延时散热机型更友好。
经验值
🧱
耐高温耐潮 + 远离热源电器位柜体 / 层板用耐高温耐潮材料（普通贴皮 / 三聚氰胺板久受热易变形开裂）；发热电器远离冰箱（贴邻加隔热板）、远离潮湿，上方不叠放重物。
高置信

诚实局限：这些数字别全信CAVEATS · 不糊弄

⚠ 这些多为厂商峰值 / 宣传口径，已标注，别为虚标付溢价
  • 蒸汽功率 2700–2865W、出汽"约 30 秒"、温控"误差 <2°C / 精控 1°C"多为峰值 / 瞬时口径或宣传话术，一律打折看
  • 美的 R6S"1 分钟蒸汽量提升 81%"、凯度"微波热效率超 50%"厂商宣传口径，缺独立验证
  • 油烟机宣称风量 24–27 m³/min、开放式厨房"排烟效率 98%"、"销量 TOP1 / 性价比之王"峰值 / 媒体评价 / 时效性排名，以第三方实测为准
  • 内胆"耐温 300°C / 850°C 烧结"、"凯度陶瓷涂层易刮花"、iQ500/300 代工与内胆材质单一评测 / 厂商说法，未独立证实，可能随批次变化

① 价格为时段口径

促销 / 补贴到手价波动大（凯度某型原始链接已失效 404），文中价格仅作锚点，以实际到手价为准。

② 间隙 / 承重多为经验值

散热顶 5–10 / 侧 2–3 / 背 3–5cm、蒸烤后部 ≥35–45mm、通风口 ≥250cm²、承重 ≥60kg 为厂商手册或安装经验值，施工以购买型号说明书为准。

③ 国标版本与限值

GB/T 17713-2022、GB/T 4706.21-2024 等已发布 / 将实施，但具体限值与实施日以现行正文为准，本篇未逐一核对正文数值。
